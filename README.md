本项目是基于Unity引擎开发的支持FireBase云同步的三消小游戏Demo

鉴于使用了Google的相关服务，所以Releases需要科学上网进行测试

本项目的可能超过100MB的稍大资源文件（PNG,MP3等）已被作者用.gitignore和.gitattributes处理过

因此如有对该项目感兴趣想要clone的佬们可能需要替换成个人的资源并配置FireBase相关文件

对于作者个人也希望几年后能看到自己的代码风格和思想能够多提升几个等级

聊以消遣，欢迎指正！
